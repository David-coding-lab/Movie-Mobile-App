# Movie-Mobile-App 🎬

A modern, high-performance mobile application for watching and streaming movies, built with **React Native** and **Expo**. This project serves as a comprehensive learning journey into mobile development, focusing on sleek UI/UX, efficient data fetching, and seamless streaming experiences.

## 🚀 Key Features

- **Movie Discovery:** Explore a vast library of trending, top-rated, and upcoming movies.
- **Search & Filter:** Easily find your favorite titles using advanced search capabilities.
- **Detailed Information:** View high-quality posters, synopses, ratings, and cast details.
- **Streaming & Watchlist:** Integration for movie playback and personalizing your watchlist.
- **Cross-Platform:** Optimized for both iOS and Android using a single codebase.

## 🛠️ Tech Stack

- **Framework:** [React Native](https://reactnative.dev/) with [Expo](https://expo.dev/)
- **Navigation:** [Expo Router](https://docs.expo.dev/router/introduction/) (File-based routing)
- **Styling:** Custom [Themed Components](components/themed-view.tsx) for Dark/Light mode support.
- **Animations:** [React Native Reanimated](https://docs.swmansion.com/react-native-reanimated/) for smooth UI transitions.
- **Icons:** Expo Symbols and Vector Icons.
- **Type Safety:** TypeScript for robust and maintainable code.

## 📁 Project Structure

```text
app/             # Expo Router directories (Tabs, Modals, etc.)
assets/          # Static assets (Images, Fonts)
components/      # Reusable UI components
constants/       # App constants and Theme configurations
hooks/           # Custom React hooks (Theme, Color Scheme)
```

## 🚥 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (LTS)
- [Expo Go](https://expo.dev/go) app on your mobile device (to test)

### Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/David-coding-lab/Movie-Mobile-App.git
   cd Movie-Mobile-App
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npx expo start
   ```

### Running on Devices

- **Android:** Press `a` in the terminal to open in an emulator or connect a device.
- **iOS:** Press `i` to open in the iOS simulator.
- **Web:** Press `w` to open in a web browser.
